📖 AI Agent 学习路线

从零基础到独立开发 AI Agent，预计学习周期 2-3 个月

01

🐍 Python 基础

⏱ 约2周 入门

Python 是 AI 开发最主流的语言，学好 Python 是一切的基础。

  • 变量与数据类型
  • 条件判断与循环
  • 函数定义与调用
  • 列表、字典、元组
  • 面向对象编程
  • 异常处理与文件操作
📖 开始学习 🧪 章节测验
02

🤖 AI 基本概念

⏱ 约1周 入门

了解 AI 的基本概念，理解大语言模型的工作原理，学会写好 Prompt。

  • 什么是机器学习
  • 大语言模型原理
  • Transformer 简介
  • Prompt 工程基础
  • Token 与上下文
  • 主流模型对比
📖 开始学习 🧪 章节测验
03

🔗 LLM API 调用

⏱ 约1周 进阶

学会用代码调用大语言模型 API，构建你的第一个 AI 对话应用。

  • OpenAI API 入门
  • API Key 安全管理
  • Chat Completion
  • 多轮对话实现
  • Function Calling
  • 成本控制技巧
📖 开始学习 💻 代码演示
04

⚙️ Agent 核心原理

⏱ 约2周 进阶

深入理解 AI Agent 的核心机制，掌握 ReAct、工具调用、记忆管理等关键技术。

  • 什么是 AI Agent
  • ReAct 推理框架
  • 工具调用与 Function Calling
  • 记忆管理系统
  • 任务规划能力
  • 多 Agent 协作
📖 开始学习 💻 代码演示
05

🛠️ Agent 框架实战

⏱ 约3周 高级

使用主流 Agent 框架快速构建应用，掌握 LangChain、AutoGen 等工具。

  • LangChain 入门
  • Chain 链式调用
  • RAG 检索增强
  • AutoGen 多智能体
  • CrewAI 协作框架
  • Dify 无代码平台
📖 开始学习 💻 代码演示
06

🚀 项目实战

⏱ 约2周 高级

综合运用所学知识，完整开发一个 AI Agent 项目并部署上线。

  • AI 搜索助手开发
  • AI 代码助手开发
  • 数据分析 Agent 开发
  • API 接口封装
  • Streamlit Cloud 部署
  • 项目总结与展示
