🔍 项目一：AI 搜索助手 入门

📌 项目概述

构建一个可以联网搜索信息的 AI Agent，用户提问，Agent 自动搜索、总结并回答。

🎯 核心功能

1. 联网搜索自动调用搜索 API 获取最新信息
2. 智能总结将搜索结果整理成清晰答案
3. 多源对比同时搜索多个来源，交叉验证
4. 追问扩展用户可追问，Agent 深入搜索

💻 核心代码

import os from langchain_openai import ChatOpenAI from langchain_community.tools import DuckDuckGoSearchRun from langchain.agents import initialize_agent, Tool # 1. 初始化模型和搜索工具 llm = ChatOpenAI(model="gpt-4o-mini", temperature=0.3) search = DuckDuckGoSearchRun() tools = [ Tool(name="search", func=search.run, description="搜索互联网获取最新信息") ] # 2. 创建 Agent agent = initialize_agent( tools=tools, llm=llm, agent="zero-shot-react-description", verbose=True ) # 3. 运行！ print("=== AI 搜索助手 ===") while True: query = input("\n你问（输入 q 退出）：") if query == 'q': break result = agent.run(query) print(f"\n答案：{result}")

  • 添加"学术搜索"专门模式
  • 加语音输入（语音转文字）
💻 项目二：AI 代码助手 进阶

📌 项目概述

构建一个可以帮用户写代码、调试代码、解释代码的 AI 助手，支持 Python 代码执行。

🎯 核心功能

1. 代码生成根据自然语言描述生成代码
2. 代码执行自动运行代码并返回结果
3. 错误调试分析报错信息，给出修复建议
4. 代码解释逐行解释代码逻辑
5. 代码优化提供性能/可读性优化建议

💻 核心代码

import subprocess from langchain_openai import ChatOpenAI from langchain.agents import tool llm = ChatOpenAI(model="gpt-4o-mini") @tool def execute_python(code: str) -> str: """执行 Python 代码并返回输出""" try: result = subprocess.run( ["python", "-c", code], capture_output=True, text=True, timeout=10 ) return result.stdout if result.returncode == 0 else f"错误：{result.stderr}" except Exception as e: return f"执行异常：{e}" @tool def explain_code(code: str) -> str: """解释 Python 代码的功能""" prompt = f"请详细解释以下 Python 代码的功能和逻辑：\n\n{code}" return llm.invoke(prompt) tools = [execute_python, explain_code] # 构建 Agent（手动 ReAct） from langchain.agents import initialize_agent agent = initialize_agent(tools, llm, agent="zero-shot-react-description", verbose=True) # 测试 result = agent.run("写一个冒泡排序，然后解释它，再执行看看结果") print(result)
⚠️ 安全注意：自动执行代码有安全风险！生产环境务必做好沙箱隔离，不要执行用户上传的恶意代码。
📊 项目三：数据分析 Agent 高级

📌 项目概述

构建一个可以自动分析数据文件（CSV/Excel）、生成图表、撰写分析报告的智能助手。

🎯 核心功能

1. 数据读取自动解析 CSV、Excel、JSON 文件
2. 数据清洗处理缺失值、异常值、重复数据
3. 统计分析计算均值、方差、相关性等
4. 可视化自动生成柱状图、折线图、饼图
5. 报告生成撰写专业数据分析报告

💻 核心代码

import pandas as pd import matplotlib.pyplot as plt from langchain_openai import ChatOpenAI from langchain.agents import tool, initialize_agent llm = ChatOpenAI(model="gpt-4o-mini", temperature=0.3) @tool def load_data(file_path: str) -> str: """加载 CSV 或 Excel 文件""" if file_path.endswith('.csv'): df = pd.read_csv(file_path) elif file_path.endswith(('.xls', '.xlsx')): df = pd.read_excel(file_path) else: return "不支持的文件格式" return f"数据加载成功！共{len(df)}行，{len(df.columns)}列\n列名：{list(df.columns)}" @tool def analyze_data(file_path: str, analysis_type: str) -> str: """数据分析""" df = pd.read_csv(file_path) num_cols = df.select_dtypes(include='number').columns if analysis_type == "统计": return str(df[num_cols].describe()) elif analysis_type == "缺失": return str(df.isnull().sum()) elif analysis_type == "相关性": return str(df[num_cols].corr()) return "未知的分析类型" @tool def visualize(file_path: str, chart_type: str, x_col: str, y_col: str): """生成可视化图表""" df = pd.read_csv(file_path) plt.figure(figsize=(10, 6)) if chart_type == "柱状图": df.plot(kind='bar', x=x_col, y=y_col) elif chart_type == "折线图": df.plot(kind='line', x=x_col, y=y_col) elif chart_type == "散点图": df.plot(kind='scatter', x=x_col, y=y_col) plt.savefig('chart.png') plt.close() return "图表已保存为 chart.png" tools = [load_data, analyze_data, visualize] agent = initialize_agent(tools, llm, agent="zero-shot-react-description", verbose=True) result = agent.run("分析 data/sales.csv 文件，查看基本统计信息并生成柱状图") print(result)
