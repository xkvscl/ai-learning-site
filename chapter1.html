🐍 第一章：Python 基础

为 AI 开发打好坚实的编程基础

⏱ 预计学习时间：2周 📊 难度：入门 ✅ 建议：先完成本章再继续

📌 1.1 变量与数据类型

变量是存储数据的"盒子"，Python 不需要提前声明类型，赋值即创建。

基本数据类型

类型示例说明
int10整数
float3.14小数
str"Hello"字符串
boolTrue / False布尔值
list[1, 2, 3]列表（可变）
tuple(1, 2, 3)元组（不可变）
dict{"name": "AI"}字典
# 变量赋值示例 name = "小明" # 字符串 age = 18 # 整数 height = 1.75 # 浮点数 is_student = True # 布尔值 print(f"我叫{name}，今年{age}岁，身高{height}m")
💡 提示：Python 中变量名建议用小写，多个单词用下划线连接，如 user_name、ai_model

🔀 1.2 条件判断

条件判断让程序根据不同情况做出不同反应，是编程的基础逻辑。

# 简单条件判断 temperature = 30 if temperature > 35: print("太热了！注意防暑") elif temperature > 25: print("天气不错，适宜活动") else: print("天气有点凉，记得添衣")
⚠️ 注意：Python 用 缩进（通常4个空格）来划分代码块，不像其他语言用花括号。这点是新手最常出错的地方！
# 多个条件判断 age = 20 has_ticket = True if age >= 18 and has_ticket: print("可以入场！") elif age < 18: print("未满18岁，禁止入场") else: print("需要门票才能入场")

🔄 1.3 循环

循环让我们不用重复写代码，可以自动处理大量数据。

for 循环 — 遍历已知次数

# 打印0到4 for i in range(5): print(f"第 {i} 次循环") # 遍历列表 fruits = ["苹果", "香蕉", "橙子"] for fruit in fruits: print(f"我喜欢吃: {fruit}")

while 循环 — 满足条件时持续执行

# while 循环 count = 0 while count < 3: print(f"计数: {count}") count += 1 # 重要：避免无限循环 print("循环结束！")
💡 实用技巧：在 AI 开发中经常需要遍历数据、处理批量任务，for 循环是最常用的！

📦 1.4 函数

函数是可重复使用的代码块，给它输入，它返回输出。就像一个小型加工厂。

# 定义一个简单函数 def greet(name): """打招呼函数""" return f"你好，{name}！欢迎学习AI！" # 调用函数 message = greet("小明") print(message) # 输出：你好，小明！欢迎学习AI！

带默认参数的函数

# 参数默认值 def chat_with_ai(user_message, model="gpt-3.5"): print(f"用户说: {user_message}") print(f"使用的模型: {model}") return f"AI回复: 收到！" # 调用 chat_with_ai("你好") # 使用默认模型 chat_with_ai("你好", model="claude") # 指定模型

返回多个值

# 返回多个值（实际返回的是元组） def get_ai_info(): name = "ChatGPT" version = "5.0" return name, version ai_name, ai_version = get_ai_info() print(f"AI名称: {ai_name}, 版本: {ai_version}")

📋 1.5 列表与字典

列表 list — 有序集合

# 创建和操作列表 ai_tools = ["ChatGPT", "Claude", "Gemini", "Llama"] # 添加元素 ai_tools.append("文心一言") print(f"工具数量: {len(ai_tools)}") # len() 获取长度 # 列表切片 print(ai_tools[0:2]) # 前两个 print(ai_tools[-1]) # 最后一个 # 列表推导式（AI开发中非常常用！） squares = [x**2 for x in range(5)] print(squares) # [0, 1, 4, 9, 16]

字典 dict — 键值对

# 字典存储结构化信息 ai_model = { "name": "GPT-4", "company": "OpenAI", "release_year": 2023, "capabilities": ["对话", "代码", "分析"] } # 访问和操作 print(ai_model["name"]) # 获取值 ai_model["context_length"] = 128000 # 添加新键值 print(ai_model.get("price", "免费")) # 安全获取（不存在返回默认值） # 遍历字典 for key, value in ai_model.items(): print(f"{key}: {value}")
💡 为什么重要：AI API 的请求和响应都是用字典（JSON格式）组织的！学会操作字典是调用 AI API 的基础。

🏗️ 1.6 类与面向对象编程

类是创建"对象"的蓝图。AI Agent、对话机器人等都用类来组织代码。

# 定义一个简单的 AI 对话类 class AIAssistant: def __init__(self, name, model): """初始化方法，创建对象时自动调用""" self.name = name self.model = model self.conversation_history = [] # 记忆 def chat(self, user_message): """处理用户消息""" print(f"【{self.name}】收到: {user_message}") self.conversation_history.append({"role": "user", "content": user_message}) # 模拟AI回复 reply = f"我是{self.name}，我正在使用{self.model}模型" self.conversation_history.append({"role": "assistant", "content": reply}) return reply def get_history(self): """获取对话历史""" return self.conversation_history # 使用类创建对象 assistant = AIAssistant(name="小智", model="GPT-4") response = assistant.chat("你好！") print(f"AI回复: {response}") print(f"对话历史: {assistant.get_history()}")
🎯 AI开发中的应用：大部分 AI 框架（LangChain、AutoGen）都大量使用类和对象。理解类的基础能帮你更好地看懂这些框架的代码！

🛡️ 1.7 异常处理

程序运行时难免出错，用 try-except 捕获异常，让程序不会崩溃。

# 异常处理示例 try: result = 10 / 0 # 这里会报错（除零错误） except ZeroDivisionError: print("错误：不能除以零！") except Exception as e: print(f"发生了其他错误: {e}") else: print("没有错误，正常执行") finally: print("无论如何都会执行这段代码")
💡 重要提示：调用 AI API 时网络可能不稳定，学会异常处理能让你的程序更稳定！

📁 1.8 文件操作

AI 应用经常需要读取/保存数据，比如保存对话记录、读取配置文件。

# 写入文件 with open("my_notes.txt", "w", encoding="utf-8") as f: f.write("这是我学Python的第一天\n") f.write("今天学了变量、循环和函数\n") # 读取文件 with open("my_notes.txt", "r", encoding="utf-8") as f: content = f.read() print(content) # 追加内容 with open("my_notes.txt", "a", encoding="utf-8") as f: f.write("继续学习中...\n")
# 读取JSON文件（AI开发中最常用） import json # 模拟AI的响应数据 ai_response = { "model": "gpt-4", "choices": [ {"message": {"role": "assistant", "content": "你好！我是AI助手"}} ] } # 保存为JSON with open("response.json", "w", encoding="utf-8") as f: json.dump(ai_response, f, ensure_ascii=False, indent=2) # 读取JSON with open("response.json", "r", encoding="utf-8") as f: loaded = json.load(f) print(loaded["choices"][0]["message"]["content"])
