🔧 环境配置

如何获取 OpenAI API Key？

+
获取步骤：
1. 访问 platform.openai.com/api-keys
2. 注册/登录 OpenAI 账号（需要海外手机号验证）
3. 点击 "Create new secret key"
4. 复制并妥善保存 Key（只显示一次！）

注意：Key = 银行卡密码，绝不能分享给他人或提交到 GitHub！

💻 技术问题

API 调用失败了怎么办（RateLimitError）？

+
RateLimitError 表示请求过快，触发了 API 的限流。解决方法：

1. 等待重试：等几秒再试
2. 加延迟：在代码里加 time.sleep(1)
3. 降低频率：不要短时间内大量请求
4. 升级账号：付费账号限流更宽松
5. 换用免费额度更多的模型：如 Claude Haiku

课程第三章有详细的错误处理代码示例。

Token 是什么？怎么计算花了多少钱？

+
Token 是 AI 模型处理文本的基本单位。
- 英文：约 4 个字符 ≈ 1 个 Token
- 中文：约 1-2 个汉字 ≈ 1 个 Token

API 费用 = 输入 Token 数 × 输入单价 + 输出 Token 数 × 输出单价

查看消耗：在 API 响应中有 usage.total_tokens 字段
控制成本：用 max_tokens 限制输出长度，开发阶段用小模型（如 GPT-4o-mini）

GitHub Pages 是什么？如何让网站上线？

+
GitHub Pages 是 GitHub 提供的免费静态网站托管服务。

启用方法（课程中已操作）：
1. 打开 GitHub 仓库 → Settings
2. 左侧点击 Pages
3. Source 选择 main 分支和 / (root)
4. 点击 Save，约 1-3 分钟网站上线

你的网站地址：https://xkvscl.github.io/ai-learning-site
只要仓库有 index.html 文件，就能在网上访问！
