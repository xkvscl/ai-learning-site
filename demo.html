🔗 LLM API 调用示例

openai_chat.py — 调用 OpenAI API 
# 安装依赖：pip install openai
import openai

# 配置 API Key（建议使用环境变量）
openai.api_key = "sk-xxxxxxxxxxxxxxxx"

def chat_with_gpt(message):
    response = openai.ChatCompletion.create(
        model="gpt-3.5-turbo",
        messages=[
            {"role": "system", "content": "你是一个有帮助的AI助手"},
            {"role": "user", "content": message}
        ],
        temperature=0.7  # 控制随机性
    )
    return response.choices[0].message.content

# 调用示例
result = chat_with_gpt("什么是AI Agent？")
print(result)
📝 代码说明：这段代码演示了如何用 OpenAI API 发送对话请求。messages 列表包含系统提示和用户消息， model 指定使用的模型，temperature 控制回答的创意程度（0-1，越高越随机）。
⚠️ 注意：实际使用时，请将 API Key 存放在环境变量中，不要直接写在代码里！
multi_turn_chat.py — 多轮对话 
# 多轮对话示例
import openai

openai.api_key = "sk-xxxxxxxxxxxxxxxx"

# 维护对话历史
messages = [
    {"role": "system", "content": "你是一个Python编程老师"}
]

def ask(user_input):
    messages.append({"role": "user", "content": user_input})
    response = openai.ChatCompletion.create(
        model="gpt-3.5-turbo",
        messages=messages
    )
    reply = response.choices[0].message.content
    messages.append({"role": "assistant", "content": reply})
    return reply

# 第一轮对话
print(ask("Python中列表和元组的区别是什么？"))
# 第二轮（Agent记得之前的上下文）
print(ask("那字典呢？和列表又有什么区别？"))
📝 代码说明：多轮对话的关键是把每次 Agent 的回复也加入 messages 列表， 这样每次请求时都带上完整的对话历史，Agent 就能"记住"之前的对话内容了。

⚙️ Agent 核心机制示例

simple_agent.py — 最简 Agent（ReAct循环） 
# 最简 Agent 实现 — ReAct 循环
import openai

class SimpleAgent:
    def __init__(self, tools):
        self.tools = tools
        self.messages = []

    def step(self, user_input):
        self.messages.append({"role": "user", "content": user_input})

        for _ in range(5):  # 最多思考5轮
            response = openai.ChatCompletion.create(
                model="gpt-3.5-turbo",
                messages=self.messages + [{
                    "role": "system",
                    "content": f"可用工具: {list(self.tools.keys())}"
                }]
            )
            msg = response.choices[0].message
            self.messages.append(msg)

            # 解析工具调用
            if msg.content.startswith("ACTION:"):
                tool_name = msg.content.split(":")[1].strip()
                if tool_name == "FINISH":
                    return msg.content.split("FINISH:")[1].strip()
                result = self.tools[tool_name]()
                self.messages.append({
                    "role": "user",
                    "content": f"工具 '{tool_name}' 返回: {result}"
                })
            else:
                return msg.content

# 使用示例
agent = SimpleAgent({
    "search": lambda: "搜索结果: AI Agent 是...",
    "calculate": lambda: "计算结果: 42"
})
print(agent.step("帮我搜索什么是AI Agent"))
📝 代码说明：这是一个最简 Agent 的核心逻辑： Thought → Action → Observation 循环。
① Agent 思考（Thought）当前任务
② 决定是否调用工具（Action）
③ 获取工具返回结果（Observation）
④ 循环直到任务完成，输出 FINISH

🛠️ LangChain 框架示例

langchain_demo.py — LangChain 基本用法 
# 安装：pip install langchain langchain-openai
from langchain_openai import ChatOpenAI
from langchain.agents import load_tools, initialize_agent

# 初始化模型
llm = ChatOpenAI(
    model="gpt-3.5-turbo",
    openai_api_key="sk-xxxxxxxx"
)

# 加载内置工具（搜索、计算器等）
tools = load_tools(["search", " Calculator"], llm=llm)

# 创建 Agent
agent = initialize_agent(
    tools=tools,
    llm=llm,
    agent="zero-shot-react-description",  # ReAct 模式
    verbose=True  # 显示思考过程
)

# 运行 Agent
result = agent.run("2024年奥运会在哪里举办？")
print(result)
📝 代码说明：LangChain 封装了 Agent 开发的大量重复工作： load_tools 加载内置工具，initialize_agent 初始化 Agent，zero-shot-react-description 是最常用的 ReAct 推理模式。设置 verbose=True 可以看到 Agent 的完整思考过程，方便调试。
