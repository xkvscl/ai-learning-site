🤖 第二章：AI 基本概念

理解人工智能的核心原理，为 Agent 开发打好理论基础

⏱ 预计学习时间：1周 📊 难度：入门

🧠 什么是人工智能？

人工智能（AI）是指让机器具有类似人类智能的能力，包括学习、推理、理解语言、识别图像等。简单来说，就是让电脑学会"思考"。

→
🗣️ 生成内容

🔤 什么是大语言模型（LLM）？

大语言模型（Large Language Model）是经过海量文本数据训练的人工智能模型，能够理解和生成人类语言。GPT、Claude、Gemini 都是 LLM。

🔑 核心原理：LLM 的本质是"文本预测机器"。它学习了互联网上几乎所有文字，学会了根据前文预测下一个词该写什么。输入一段文字，它预测下一段应该是什么。

Token — 模型的"词汇单位"

模型不像我们一样按"字"或"词"处理文本，而是按 Token 处理：

语言Token 估算说明
英文约4个字符 ≈ 1个Token简单单词可能1个Token
中文约1-2个汉字 ≈ 1个Token每个字约1-2个Token
代码依赖具体内容可能比英文更紧凑

所以 GPT-4 的 128K 上下文窗口，大约能放 10万字的中文文章！

⚡ Transformer 架构 — GPT 的心脏

2017 年 Google 发表的论文《Attention Is All You Need》提出了 Transformer 架构，这是现代 AI 的基石。

🔄 Transformer 处理流程

输入文字
→
分词(Tokenize)
→
嵌入(Embed)
→
注意力机制
→
输出文字

注意力机制（Attention）— 理解上下文的关键

Attention 让模型能够"关注"输入中最重要的部分。比如读这句话：

"小明把苹果**吃掉**了，因为这个**苹果**很甜。"

第一个"苹果"指的是**水果**，第二个"苹果"是**公司名称**。Attention 机制帮助模型理解这种上下文关系！

💡 记住：Transformer 解决了之前 RNN（循环神经网络）无法处理长文本的问题，是 AI 发展的关键突破！

💬 Prompt 工程 — 和 AI 高效对话的技巧

Prompt 就是你发给 AI 的指令。好的 Prompt 能让 AI 输出更准确、更实用的答案。

核心原则

原则❌ 差的做法✅ 好的做法
明确具体"帮我写代码""用 Python 写一个函数，输入列表返回平均值"
指定角色"解释一下量子计算""你是一位物理教授，用通俗语言解释量子计算"
给出例子"写一首诗""写一首七言绝句，主题是春天，风格仿照《春晓》"
分步骤"帮我做数据分析""1. 先分析数据分布；2. 找出异常值；3. 做可视化"

常用 Prompt 技巧

# 技巧1：Few-shot（给例子） prompt = """ 请判断以下评论的情感是正面还是负面。 示例： 评论："这个产品太差了，完全不能用" 情感：负面 请判断： 评论："终于解决了困扰我一周的问题！" 情感：""" # 技巧2：思维链（Chain of Thought） prompt = """ 问题：小明有5个苹果，小红给了他3个，他又吃了2个，还剩几个？ 让我们一步一步思考：""" # 技巧3：指定输出格式 prompt = "用JSON格式返回：{\"name\": 姓名, \"age\": 年龄, \"city\": 城市}"}

⚙️ 常见模型参数详解

Temperature — 随机性控制

值效果适用场景
0.0几乎确定性输出，同一个问题答案固定精确问答、代码生成
0.3-0.5略有变化，保持相关性日常对话、总结
0.7-1.0高随机性，更多创意创意写作、脑暴

Top-p / Nucleus Sampling — 采样范围

控制模型从多大范围内选择下一个词。Top-p=0.9 意味着只从概率最高、累计达90%的词中选。

Max Tokens — 最大输出长度

限制模型输出的最大 Token 数。设置太小可能截断，设置太大会浪费资源。

System Prompt / System Message

# System Prompt 设置AI的角色和行为 messages = [ {"role": "system", "content": "你是一位Python编程老师，说话亲切，喜欢用例子解释概念"}, {"role": "user", "content": "什么是递归？"} ] # 用户的每条消息都在同一角色下继续

🤖 AI Agent 和普通 AI 对话的区别

普通 AI 对话 = 问答（你问我答）
AI Agent = 自主行动（你给我目标，我自己完成）

能力普通 AI 对话AI Agent
回答问题✅ 可以✅ 可以
调用工具（搜索/代码）❌ 不行✅ 可以
多步骤任务❌ 一次性回答✅ 规划+执行
记忆历史有限✅ 可持久化记忆
访问互联网❌ 不能✅ 可以（配合工具）
执行代码❌ 不能✅ 可以（配合工具）
🎯 简单理解：普通 AI 像一本百科全书，而 AI Agent 像一个能替你工作的助手。百科全书只能告诉你答案，助手能帮你完成任务！
