🐍 Python 基础测验 第1章

Q1. Python 中，用什么关键字来定义函数？

A. function
B. def
C. func
D. define
✅ 正确！Python 使用 def 关键字定义函数，语法为 def 函数名(参数):
❌ 答错了！Python 使用 def 关键字来定义函数，而不是其他选项。

Q2. 以下哪种数据结构在 Python 中是不可变的（Immutable）？

A. 列表 list
B. 字典 dict
C. 元组 tuple
D. 集合 set
✅ 正确！元组（tuple）创建后内容不可修改，是不可变数据类型。
❌ 答错了！元组（tuple）是不可变的，其他选项都是可变的。

Q3. print([x*2 for x in range(3)]) 的输出结果是什么？

A. [1, 2, 3]
B. [0, 2, 4]
C. [2, 4, 6]
D. [0, 1, 2]
✅ 正确！range(3) 生成 [0,1,2]，每个元素乘以2得到 [0,2,4]。这是列表推导式的用法。
❌ 答错了！range(3) 生成 [0,1,2]，乘以2后是 [0,2,4]。

Q4. 下面哪个选项是 Python 中正确的字典访问方式？

A. person(0)
B. person[0]
C. person["name"] 或 person.get("name")
D. person->name
✅ 正确！字典用键来访问，如 person["name"]，或用安全的 person.get("name")
❌ 答错了！字典用键访问，如 person["name"]。
🤖 AI 基本概念测验 第2章

Q5. 大语言模型（LLM）中的 "Token" 是什么？

A. 一个完整的单词
B. 一个中文字符
C. 模型处理文本的基本单位
D. 一段话
✅ 正确！Token 是模型处理文本的基本单位，英文约4字符≈1 Token，中文约1-2字≈1 Token。
❌ 答错了！Token 是模型处理文本的基本单位，不是简单的"字"或"词"。

Q6. "Temperature" 参数设为 0 和 1.0，哪个输出更随机、更有创意？

A. 0
B. 1.0
C. 两者完全一样
D. 无法确定
✅ 正确！Temperature 越高，输出越随机；Temperature=0 时几乎确定，Temperature=1.0 时最有创意。
❌ 答错了！Temperature=1.0 输出最随机，Temperature=0 最确定。

Q7. Transformer 架构中的 "Attention"（注意力机制）主要解决了什么问题？

A. 让模型运行得更快
B. 减少模型参数
C. 让模型理解长文本中的上下文关系
D. 自动翻译语言
✅ 正确！Attention 让模型能够"关注"输入中最重要的部分，理解上下文关系，解决了长文本处理难题。
❌ 答错了！Attention 的核心是让模型理解上下文关系，而不只是提升速度。

Q8. AI Agent 与普通 AI 对话最核心的区别是什么？

A. Agent 的回答更准确
B. Agent 能自主规划、调用工具、执行多步骤任务
C. Agent 只能处理文字
D. Agent 不需要大语言模型
✅ 正确！Agent 的核心是自主性：能感知环境、制定计划、调用工具，完成复杂的多步骤任务。
❌ 答错了！Agent 的核心是自主规划和使用工具，不只是更好的回答。

Q9. ReAct 框架中，"ReAct" 代表什么？

A. Real Action（真实行动）
B. Reasoning + Acting（推理 + 行动）
C. Reactive Agent（反应式智能体）
D. Recursive Action（递归行动）
✅ 正确！ReAct = Reasoning（推理）+ Acting（行动）。核心循环：Thought → Action → Observation → ...
❌ 答错了！ReAct = Reasoning（推理）+ Acting（行动），是思考和行动的结合。

C. 情景记忆（经验教训）
D. 视觉记忆（图片存储）

❌ 答错了！Agent 记忆分为短期、长期和情景三类，"视觉记忆"不是标准分类。
