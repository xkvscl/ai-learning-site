🐍 Python 基础测验

Q1. 以下哪个是 Python 中定义函数的正确方式？

A. function greet(name):
B. def greet(name):
C. func greet(name):
D. define greet(name):
✅ 正确答案是 B。Python 使用 def 关键字定义函数，语法为 def 函数名(参数):

Q2. Python 中，以下哪种数据结构是不可变的（immutable）？

A. 列表 list
B. 字典 dict
C. 元组 tuple
D. 集合 set
✅ 正确答案是 C。元组（tuple）创建后不能修改，是不可变数据结构。列表、字典、集合都是可变的。

Q3. 以下代码的输出结果是什么？
print([x*2 for x in range(3)])

A. [1, 2, 3]
B. [0, 2, 4]
C. [2, 4, 6]
D. [0, 1, 2]
✅ 正确答案是 B。range(3) 生成 [0,1,2]，每个元素乘以2得到 [0,2,4]。这是列表推导式的用法。

🤖 AI 基本概念测验

Q4. 大语言模型（LLM）中的 "Token" 是指什么？

A. 一个完整的单词
B. 一个字符
C. 模型处理文本的基本单位，可能是词、词片段或标点
D. 一行文字
✅ 正确答案是 C。Token 是模型处理文本的基本单位，英文中大约 4 个字符 = 1 个 Token，中文大约 1-2 个字 = 1 个 Token。

Q5. Prompt 工程中，"Temperature" 参数的作用是什么？

A. 控制模型的运行速度
B. 控制输出的随机性，值越高输出越随机有创意
C. 控制输出的最大长度
D. 控制模型的记忆能力
✅ 正确答案是 B。Temperature 控制输出的随机性：0 表示确定性输出，1 表示较随机，适合创意写作；精确任务建议用低值。

⚙️ AI Agent 测验

Q6. AI Agent 与普通 LLM 对话的最大区别是什么？

A. Agent 的回答更准确
B. Agent 只能处理文字
C. Agent 能自主规划、调用工具、执行多步骤任务
D. Agent 不需要大语言模型
✅ 正确答案是 C。AI Agent 的核心特征是自主性：能感知环境、制定计划、调用外部工具（搜索、代码执行等），完成复杂的多步骤任务。

Q7. ReAct 框架中，"ReAct" 代表什么？

A. Real Action（真实行动）
B. Reasoning + Acting（推理 + 行动）
C. Reactive Agent（反应式智能体）
D. Recurrent Action（循环行动）
✅ 正确答案是 B。ReAct = Reasoning（推理）+ Acting（行动）。Agent 先思考（Thought），再行动（Action），再观察结果（Observation），循环直到完成任务。
