🛠️ 第五章：Agent 框架实战

使用 LangChain、AutoGen、CrewAI 等主流框架，快速构建 AI Agent 应用

⏱ 预计学习时间：3周 📊 难度：高级

⛓️ 5.1 LangChain 入门

LangChain 是最流行的 AI Agent 开发框架，封装了大量常用组件，让你不用从零写代码。

安装 LangChain

# 基础安装 pip install langchain langchain-openai # 常用扩展 pip install langchain-community # 第三方工具集成 pip install langchainhub # Prompt 模板市场 pip install faiss-cpu # 向量数据库（用于 RAG）

第一个 LangChain Agent

# langchain_first_agent.py from langchain_openai import ChatOpenAI from langchain.agents import load_tools, initialize_agent # 1. 初始化大模型 llm = ChatOpenAI( model="gpt-4o-mini", temperature=0.7, api_key="sk-xxxx" # 建议用环境变量 ) # 2. 加载工具（搜索 + 计算器） tools = load_tools(["search", "ddg-search", "Calculator"], llm=llm) # 3. 初始化 Agent（使用 ReAct 模式） agent = initialize_agent( tools=tools, llm=llm, agent="zero-shot-react-description", # ReAct 模式 verbose=True, # 打印思考过程 max_iterations=5 # 最多思考5轮 ) # 4. 运行！ result = agent.run("2025年诺贝尔物理学奖得主是谁？") print(result)
💡 verbose=True 非常有用！开启后你可以看到 Agent 的完整思考过程，理解它为什么做出每个决定。

🔗 5.2 LangChain Chain（链式调用）

Chain 是 LangChain 的核心概念——把多个步骤串联起来工作。

LCEL — LangChain Expression Language

# 用 LCEL 构建 Chain（管道式写法） from langchain_openai import ChatOpenAI from langchain.prompts import PromptTemplate from langchain.output_parsers import StrOutputParser llm = ChatOpenAI(model="gpt-4o-mini") # 链式调用 chain = ( PromptTemplate.from_template("请用一句话介绍：{topic}") | llm | StrOutputParser() ) # 执行 result = chain.invoke({"topic": "人工智能"}) print(result) # 输出：一句话介绍

构建 RAG 问答系统

# RAG = Retrieval Augmented Generation（检索增强生成） # 原理：先从文档中检索相关内容，再让 AI 基于检索结果回答 from langchain_openai import OpenAIEmbeddings, ChatOpenAI from langchain_community.vectorstores import FAISS from langchain.text_splitter import RecursiveCharacterTextSplitter from langchain.chains import RetrievalQA # 1. 准备文档 documents = [ "Python的优势：易学、功能强大、生态丰富...", "Java的优势：跨平台、企业级应用、生态成熟...", "JavaScript的优势：全栈开发、前端唯一选择..." ] # 2. 分块 + 向量化 splitter = RecursiveCharacterTextSplitter(chunk_size=100, chunk_overlap=20) chunks = splitter.split_text("\n".join(documents)) # 3. 存入向量数据库 embeddings = OpenAIEmbeddings() vectorstore = FAISS.from_texts(chunks, embeddings) # 4. 构建 RAG Chain qa_chain = RetrievalQA.from_chain_type( llm=ChatOpenAI(model="gpt-4o-mini"), retriever=vectorstore.as_retriever() ) # 5. 问答 result = qa_chain.run("Python和Java哪个更适合初学者？") print(result)

🤝 5.3 AutoGen — 微软多智能体框架

AutoGen 是微软开源的多智能体协作框架，多个 Agent 可以对话、协作完成复杂任务。

# 安装 pip install autogen import autogen # 1. 定义 Agent（类似定义角色） assistant = autogen.AssistantAgent( name="assistant", llm_config={ "model": "gpt-4o-mini", "api_key": "sk-xxxx", "temperature": 0.7 } ) # 2. 定义用户代理（代表人类） user_proxy = autogen.UserProxyAgent( name="user_proxy", human_input_mode="NEVER", # 不需要人工介入 max_consecutive_auto_reply=3 ) # 3. 启动对话 user_proxy.initiate_chat( assistant, message="帮我用Python写一个快速排序算法，并测试它" )

多 Agent 协作示例

# 多 Agent 协作：代码审查流水线 import autogen # Agent 1: 开发者 developer = autogen.AssistantAgent( name="Developer", llm_config={"model": "gpt-4o-mini", "temperature": 0.3}, system_message="你是Python开发工程师，负责编写代码。" ) # Agent 2: 代码审查员 reviewer = autogen.AssistantAgent( name="Reviewer", llm_config={"model": "gpt-4o-mini", "temperature": 0.3}, system_message="你是资深代码审查员，负责检查代码质量、安全和性能问题。" ) # Agent 3: 测试工程师 tester = autogen.AssistantAgent( name="Tester", llm_config={"model": "gpt-4o-mini", "temperature": 0.3}, system_message="你是测试工程师，负责验证代码正确性。" ) # 编排多 Agent 协作 groupchat = autogen.GroupChat(agents=[developer, reviewer, tester], messages=[]) manager = autogen.GroupChatManager(groupchat=groupchat) # 启动协作任务 user_proxy.initiate_chat( manager, message="写一个计算器程序，实现加减乘除，并做好测试" )

👥 5.4 CrewAI — 多 Agent 编排框架

CrewAI 是新兴的多 Agent 框架，主打"角色扮演 + 任务协作"的概念，特别适合业务流程自动化。

# 安装 pip install crewai crewai-tools from crewai import Agent, Task, Crew # 1. 定义 Agent（角色） researcher = Agent( role="高级研究员", goal="搜索并整理最准确的信息", backstory="你是一名有10年经验的研究员，擅长信息检索和分析。", verbose=True ) writer = Agent( role="专业内容编辑", goal="将研究结果转化为通俗易懂的文章", backstory="你是资深编辑，擅长把复杂内容写得易懂有趣。", verbose=True ) # 2. 定义任务 task1 = Task( description="研究 AI Agent 的最新发展趋势，整理3个重要趋势", agent=researcher ) task2 = Task( description="根据研究报告，写一篇1000字的科普文章", agent=writer, context=[task1] # 依赖前一个任务的输出 ) # 3. 组建团队，执行任务 crew = Crew(agents=[researcher, writer], tasks=[task1, task2]) result = crew.kickoff() print(result)

🌐 5.5 Dify — 开源 LLM 应用平台

Dify 是一个开源的 LLM 应用开发平台，零代码即可创建 AI 应用，适合非程序员。

特性Dify代码开发（LangChain）
编程要求零代码/低代码需要 Python 编程
灵活性一般非常高
部署难度一键部署需要自己搭建
适合人群产品/运营/非程序员开发者
费用自托管免费代码免费，API 付费

Dify 的主要功能

# Dify 可以做的事（不需要写代码）： # - 可视化搭建 RAG 流程 # - 创建 AI 对话应用 # - 定义 Agent 工作流 # - 集成多种模型（OpenAI/Claude/本地模型） # - API 一键发布
🎯 推荐学习路径：初学者可以用 Dify 快速体验 AI 应用搭建，理解 Agent 的工作原理后再学习 LangChain，从可视化工具有了直观认知，再深入代码层面。

🚀 5.6 实战项目：构建一个 AI 助手

综合运用所学知识，构建一个完整的 AI 助手项目。

项目需求

# 项目：个人 AI 学习助手 # 功能需求： # 1. 能够回答 AI 相关问题（知识库） # 2. 能够联网搜索最新信息 # 3. 能够执行 Python 代码 # 4. 能够记住用户的偏好和历史 # 5. 有友好的对话界面 # 技术选型： # - LangChain（Agent 框架） # - GPT-4o-mini（经济实惠） # - FAISS（本地向量数据库） # - Streamlit（快速搭建 Web 界面）

完整项目结构

# 项目结构 ai-assistant/ ├── app.py # Streamlit 主界面 ├── agent/ │ ├── __init__.py │ ├── assistant.py # Agent 核心逻辑 │ ├── tools.py # 工具定义 │ └── memory.py # 记忆管理 ├── data/ │ └── knowledge.md # 知识库文档 ├── requirements.txt # 依赖列表 └── README.md # 项目说明

核心代码

# agent/assistant.py from langchain_openai import ChatOpenAI from langchain.agents import initialize_agent, Tool from langchain.memory import ConversationBufferMemory from agent.tools import get_tools from agent.memory import AgentMemory class AIAssistant: def __init__(self, name="小智"): self.name = name self.llm = ChatOpenAI(model="gpt-4o-mini", temperature=0.7) self.memory = AgentMemory() self.tools = get_tools(self.llm) self.agent = initialize_agent( tools=self.tools, llm=self.llm, agent="conversational-react-description", memory=self.memory.to_langchain(), verbose=True ) def chat(self, message): # 加入记忆上下文 context = self.memory.get_context() full_message = f"用户说：{message}\n\n相关记忆：{context}" response = self.agent.run(full_message) self.memory.add("user", message) self.memory.add("assistant", response) return response # app.py import streamlit as st from agent.assistant import AIAssistant st.title("🤖 我的 AI 学习助手") if "assistant" not in st.session_state: st.session_state.assistant = AIAssistant() user_input = st.text_input("你有什么问题？", "") if user_input: response = st.session_state.assistant.chat(user_input) st.markdown(f"**助手：** {response}")
